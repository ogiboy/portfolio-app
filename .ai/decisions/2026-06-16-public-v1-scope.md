# Decision: Public Portfolio v1

## Status

Accepted

## Context

The overhaul target is a public, client-first hybrid portfolio rather than an authenticated product surface.

## Decision

Ship v1 as a public portfolio. Keep all existing projects available, prioritize client-facing positioning, and treat hiring/creative proof as secondary but visible tracks.

## Consequences

- Auth0 login and private dashboard are outside v1 scope.
- Contact and project inquiry become primary conversion paths.
- EN/TR routes remain first-class.
